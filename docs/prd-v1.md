# LaunchPilot AI - MVP PRD

## Problem

Forward Deployed Engineers spend significant time manually validating AI-generated landing pages before they can be deployed.

Current workflow requires checking:

- Page loads correctly
- Mobile responsiveness
- Performance
- SEO
- Tracking pixels
- Accessibility

This process is repetitive and error-prone.

---

## Solution

LaunchPilot AI automates the validation process.

Users enter a landing page URL.

The application:

1. Opens the page.
2. Captures a screenshot.
3. Runs Lighthouse.
4. Detects tracking scripts.
5. Uses AI to review UX.
6. Generates a report.

---

## MVP Scope

Only implement:

- Dashboard
- URL Input
- Screenshot Capture
- Display Screenshot

Everything else is out of scope.
